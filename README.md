# ImageVerse

A modern image editing web application built with React. All processing happens client-side for complete privacy.

</div>

## Features

- **Image Compressor** - Reduce file size with quality control
- **Background Remover** - AI-powered background removal
- **Image Upscaler** - Enhance resolution up to 4x
- **Format Converter** - Convert between JPG, PNG, WEBP
- **Cropper & Flip** - Custom dimensions and orientation

## Tech Stack

- React 19 + Vite
- Tailwind CSS 4
- React Router 7
- browser-image-compression, JSZip

## Getting Started

```bash
# Clone repository
git clone https://github.com/yourusername/image-verse.git
cd image-verse

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
